# 我的画像 / User Profile

> 排名与「对我的影响」都基于这份画像。越具体，Top5 越准。

## 角色 / Role
AI 产品 / 创业方向的从业者。判断一篇论文时最看重「能不能、以及如何用进真实产品」，而不是纯学术新意。

## 当前在做的项目 / Current projects
- 一个基于 LLM 的**导购 / agent 产品**
- **个人论文情报与知识管理系统**（就是这个平台）
- 持续为**技术选型与决策**跟踪前沿进展

## 重点关注方向 / Focus areas（排名时加权）
- **LLM agent、工具调用（tool-use）、记忆与技能管理**（SkillOS / 技能库 类）
- **多模态、推理（reasoning）模型、新架构**
- **大厂产品与政策动向**（OpenAI / Anthropic / Google 的发布、监管、竞争格局信号）

## 次要 / 顺带关注 / Secondary
- RAG、检索、知识图谱（导购域会用到，但不是第一优先）
- 训练 / 推理效率（够用即可，不追底层细节）

## 不太关心 / Low interest
- 纯理论、与落地无关的数学推导
- 与上述方向无关的细分领域（纯硬件、纯生物信息、纯底层 CV 等）

## 「对我的影响」意味着什么 / What "impact on me" means
对每篇论文，我想知道：
1. 它能不能、以及如何用进我的**导购/agent 产品**或这个**情报平台**；
2. 它改变了我哪个原本的判断或做法（尤其是技术选型层面）；
3. 我接下来该做什么（读全文 / 动手试 / 持续关注后续）。
