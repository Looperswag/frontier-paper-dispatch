# 我的画像 / User Profile（模板）

> 复制本文件为 `config/profile.md` 再按你的真实情况改。
> 排名（Top5）与每篇的「对我的影响」都基于这份画像——**越具体越准**。

## 角色 / Role
（示例）AI 产品 / 创业方向的从业者。判断一篇论文时你最看重什么？比如「能不能落地到产品」「方法是否新颖」「商业/政策信号」。

## 当前在做的项目 / Current projects
- （示例）一个基于 LLM 的 ____ 产品
- （示例）在搭建的 ____ 系统 / 在做的 ____ 研究

## 重点关注方向 / Focus areas（排名时加权）
- （示例）LLM agent、工具调用（tool-use）、记忆与技能管理
- （示例）RAG / 检索 / 多模态 / 推理模型 / 训练与推理效率…只留你**真正**关心的

## 不太关心 / Low interest
- （示例）纯理论、与落地无关的数学推导
- （示例）与你方向无关的细分领域

## 「对我的影响」意味着什么 / What "impact on me" means
对每篇论文，你想让 AI 回答的问题，例如：
1. 它能不能、以及如何用进我的项目；
2. 它改变了我哪个原本的判断或做法；
3. 我接下来该做什么（读全文 / 动手试 / 持续关注）。

> 提示：用一阵子后，`npm run refine` 会根据你的批注和提问，自动给出精炼建议（见 README「自进化飞轮」）。
