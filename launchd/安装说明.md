# 定时任务安装（每晚 22:00 自动推送）

plist 里已经写好你的绝对路径，直接装即可。

## 安装
```bash
cp "/Users/ali-jason/前沿论文情报台/launchd/com.frontierpapers.ingest.plist" ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.frontierpapers.ingest.plist
```

## 立即测一次（不用等到 22:00）
```bash
launchctl start com.frontierpapers.ingest
# 看日志（DeepSeek 跑几分钟）：
tail -f "/Users/ali-jason/前沿论文情报台/ingest.log"
```
看到 `已推送到微信 ✉️` + 微信收到，就说明定时链路 OK。

## 常用命令
```bash
launchctl list | grep frontierpapers          # 看是否已加载
launchctl unload ~/Library/LaunchAgents/com.frontierpapers.ingest.plist   # 停用
# 改了 plist 后要先 unload 再 load 才生效
```

## 注意
- launchd 只在 **Mac 开机且未睡眠** 时触发；22:00 若在睡眠，会在下次唤醒后补跑一次。
- 想永远准点，可在「系统设置 → 电池/节能」里设 22:00 前自动唤醒，或让 Mac 那时别睡。
- 改推送时间：编辑 plist 里的 `Hour`/`Minute`，然后 unload + load。

# 每周画像精炼（可选）

每周日 23:00 跑 `npm run refine`，只生成建议 `config/profile.suggested.md`（**不**覆盖 `profile.md`）。你有空时瞄一眼，满意再 `npm run refine:apply`。

## 安装
```bash
cp "/Users/ali-jason/前沿论文情报台/launchd/com.frontierpapers.refine.plist" ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.frontierpapers.refine.plist
```
立即测一次：`launchctl start com.frontierpapers.refine`，看 `refine.log`。
卸载：`launchctl unload ~/Library/LaunchAgents/com.frontierpapers.refine.plist`
改时间：编辑 plist 里的 `Weekday`(0=周日) / `Hour` / `Minute`，再 unload + load。
